## 5. 딕셔너리 : { key, value1 : key, value2...} 
- key 값은 변함없어야 함 -> 튜플형 가능(리스트 불가) 
- key 값은 중복 불가능한 고유값 

## 6. 집합
- s1.intersection(s2) : 교집합(&)
- s1.union(s2) : 합집합 (|) 
- s1.difference(s2) : 차집합(-) 
- s1.add(4) : 값 1개 추가
- s1.update([4,5,6] ) : 값 2개이상 추가
- s1.remove(2) : 특정 값 제거
